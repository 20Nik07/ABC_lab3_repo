# Лабораторная работа №3
## Система контроля версий Git

### Цель:
Освоить базовые и продвинутые возможности Git:
- создание и фиксация изменений (commit)
- работа с ветками (branch)
- слияние (merge)
- перемещение (rebase)
- разрешение конфликтов

### Файлы:
- run.txt — основной файл лабораторной  
- diff.txt — вывод разницы между коммитами  
- history.txt — история коммитов  
- README.md — описание проекта
